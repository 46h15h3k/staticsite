A static site, hand-wired to a Linux box you control.

No platform, no build service, no magic. Just an EC2 instance, an nginx config written by hand, and a shell script that rsyncs these three files to it whenever something changes.

How this gets built

01

Reach the box over SSH

Launch a small EC2 instance, open ports 22 and 80 in its security group, and connect with a key pair — ssh -i key.pem ubuntu@<ip>.

02

Install and point nginx at this folder

Install nginx, drop a server block in /etc/nginx/sites-available/ pointing root at /var/www/site, then nginx -t and reload.

03

Push changes with rsync

Run ./deploy.sh — it rsyncs this folder to the server over SSH, only transferring what's changed.
